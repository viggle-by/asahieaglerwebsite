EaglercraftX Offline

The 1.8.8 version of Eaglercraft. Switching to this version is highly recommended from older versions as it introduces a ton of new features, blocks, bug fixes and optimizations.

Make sure to keep your Offline download in a safe location, such as a USB drive or a folder to keep it safe.

Download (zip)

EaglercraftX WASM-GC Offline

The WebAssembly GC version of 1.8.8 Eaglercraft. This version is significantly faster than the JavaScript one, however it is not supported on all browsers yet (like Safari) and currently requires access to chrome://flags to run on Google Chrome.

This will eventually replace the JavaScript build as the recommended version of Eaglercraft once more browsers support it.

Download (zip)

Eaglercraft Offline

The 1.5.2 version of Eaglercraft. If eaglercraft.com is blocked on your network, you can use this Offline download to bypass network restrictions.

Make sure to keep your Offline download in a safe location, such as a USB drive or a folder to keep it safe.

EaglerXBungee (1.3.4)

A Bungeecord plugin that runs a Websocket allowing EaglercraftX players to join your Java Edition server.

The EaglercraftX Plugin will not work on Spigot and requires a Bungeecord proxy.
